ANALYZING PAINTER’S INFLUENCES ON ARTWORKS WITH CNNS

Some more about the project

Why this project?

This project was done as the semester project in the graduate level course ME536 Design of Intelligent Machines. Why not just use Google’s image search for this? Because that will just give you visually similar images We are searching for the similar artistic style, the artistic style that the model can differentiate with an 87% accuracy.

Abstract

Treat this multiclass classifier as a Blackbox. Can be used for much more than a classifier as the features of Blackbox are that it tells how much artistic element of each artist is present in a new painting. Find overlaps of artistic elements in paintings. Exploit these overlaps to identify artistically similar pictures together.

Project Details



Meaningful Data Augmentation

What not to do?

  • Color Augmentation (Brightness, Saturation, Contrast, Hue)
  • Non Affine Transformations (Fisheye, Horizontal Wave, Warp)

Why not to do the above?

  • The nature of the data as important feature that is to be preserved through these transformations would be the “Artistic Style”
  • “Artistic Style” would not be maintained through color augmentations
  • Non Affine Transformations would lead to a whole new style altogether.


Improving the model with Meaningful Data Augmentation


What’s Left? Affine Transformations

  • Creating synthetic data with more variety from existing data

  • Affine Transformations Introduced:
    • Horizontal and Vertical Flip

    • Rotation

    • Translations ( Height Shift, Width Shift)

Data Analysis to find Similar Images

  • Used classifier on over 4000 paintings to get a distribution of each artistic element in each painting.
    • Formed a feature space matrix of 4299x11

  • Input my own painting to the model, get an output of 1x11

  • Narrow the 1 x 11 vectors to 1 x n vector where these “n” percentages sum up to 70% of the probability distribution.

  • The feature space is then reduced to these “n” columns as well to find the Euclidian distance based on the modified input and feature space


Demystifying how the CNN Model thinks

Based on the result from the previous picture:

  • What gives away a Picasso painting?
    • The eyes are not the same size

    • The eyes are not on the same elevation

    • Faces are not full frontal

  • With adding a Homer Simpson image that follows all those features, it was identified to have Picasso's artisitic element with 88% assurity.
