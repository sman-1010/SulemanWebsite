PREDICTING MUSIC GENRE CHOICE FROM PHYSICAL ACTIVITY DATA

THIS PROJECT IS A WORK IN PROGRESS

Some more about the project

Why this project?

This is a personal project I am doing for fun.

Abstract

There are two parts of this project, the first part is Data Analysis where I make my own version of Spotify Wrapped by requesting my extensive Music Play history from Spotify. The second part is attempting to find patterns and correlation between my physical activity data from my fitness band over the past year along with my music playback history of the same time.

Languages and Libraries Used

  • Python
  • TensorFlow
  • Pandas
  • Seaborn
  • Matplotlib

Project Details



Preprocessing the Data

  • As I was not in the same timezone throughout the year, the date and timing had to be adjusted to have meaningful interpretations

  • Missing information of data had to be handled

  • Songs that were played for less than half a second were removed as played songs, as these can not be considered to be a "play" or a "skip"
