JOINT SEMANTIC-GEOMETRIC LEARNING FOR POLYGONAL BUILDING SEGMENTATION

Some more about the project

Why this project?

This project was done as the semester project in the graduate level course CENG501 Deep Learning.

Abstract

This project consists of two parts which is a Multi-task Segmentation Network that predicts vertices of the polygon to segment the building, next there is a Polygon Refinement Network which takes these vertices and the cropped segment of the building and produces displacement values for each vertex to improve the polygonal segmentation

Project Details



Resnet Backbone & Vertex Embedding


The resnet backbone is a variant of the ResNet50 where skip-connection structure are used to up-sample and concatenate the feature maps of four skip layers. The size of the final feature map for vertex embedding. The input to the Resnet Backbone is 224x224x3 cropped image of the building segment from the original picture through the bounding boxes corresponding to each building instance and rescaled into the same size. The skip concatenation leads to a feature map of 112x112x256 for the same building segment. The coordinates of building vertices obtained from Vertex Generation Model are transformed accordingly for vertex embedding on the final feature map of the backbone. Each vertex is assigned with the features extracted from the channel direction of the cube.


Propagation Model & Coordinate Transforming & Refined Polygon Vertices


The polygonal vertices obtained from the Vertex Embedding step are considered as nodes of a graph, and every two neighbouring vertices constitutes an edge of the graph. A gated graph neural network is used to learn the offset needed for each vertex , the displacement needed to move the vertex to the ground truth. After the propagation process through the Gated Graph Neural Network, two fully-connected layers are used to output a displacement value for each vertex. The prediction of displacement value is also formulated as a classification issue and the model is trained with cross-entropy loss. In the coordinate transforming step, the output displacement classes of PRN are converted to the displacement coordinates, and added to the corresponding vertex coordinates.




Feature Extractor Result from Resnet Backbone

As the ResNet50 is used as a feature extractor for the building instance images, it uses skip connects compare to the classic ResNet50 and concatenates them at the end leading to a feature map of depth 256 where each feature map has dimensions of 112 x 112 pixels. On the left, six of such feature maps are shown. The size of the final feature map for vertex embedding is 112x112x256.
